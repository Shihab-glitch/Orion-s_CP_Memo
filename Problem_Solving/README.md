# Problem Solving

This folder contains solutions to problems from various competitive programming platforms.

## Subfolders

- **Codeforces/** - Solutions from Codeforces
- **LeetCode/** - Solutions from LeetCode
- **HackerRank/** - Solutions from HackerRank
- **CodeChef/** - Solutions from CodeChef
- **AtCoder/** - Solutions from AtCoder

## File Format

Each solution should include:
```
/*
 * Problem: [Problem Name]
 * Link: [Problem URL]
 * Difficulty: [Easy/Medium/Hard]
 * Tags: [Algorithm types, Data structures used]
 * 
 * Description:
 * [Brief problem description]
 * 
 * Approach:
 * [Explanation of solution approach]
 * 
 * Time Complexity: O(?)
 * Space Complexity: O(?)
 */
```
