# Data Structures

This folder contains implementations of various data structures used in competitive programming.

## Subfolders

- **Arrays_and_Strings/** - Array and string manipulation techniques
- **Linked_Lists/** - Linked list implementations and problems
- **Stacks_and_Queues/** - Stack and queue implementations
- **Trees/** - Binary trees, BST, AVL, Segment trees, etc.
- **Graphs/** - Graph representations and operations
- **Heaps/** - Min heap, max heap, priority queues
- **Hash_Tables/** - Hash map/set implementations
