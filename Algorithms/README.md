# Algorithms

This folder contains implementations of various algorithms used in competitive programming.

## Subfolders

- **Sorting/** - Sorting algorithms (Quick Sort, Merge Sort, Heap Sort, etc.)
- **Searching/** - Searching algorithms (Binary Search, Linear Search, etc.)
- **Graph_Algorithms/** - Graph traversal and algorithms (BFS, DFS, Dijkstra, Bellman-Ford, etc.)
- **Dynamic_Programming/** - DP solutions and patterns
- **Greedy/** - Greedy algorithm implementations
- **Divide_and_Conquer/** - Divide and conquer solutions

## Tips

- Include time and space complexity in comments
- Add test cases for verification
- Document the algorithm approach
